# Cloudlets

Cloudlets is a small Terraform project for provisioning KVM virtual machines on Ubuntu systems using bridged networking. It includes a simple Bash script that automates Terraform installation, manages variables, and runs the deployment.

## Requirements

•	Ubuntu operating system
•	Bridged networking set up
•	Terraform and terraform-provider-libvirt

## Features

•	Works both locally and remotely
•	Simplifies VM provisioning using Terraform
•	Automated setup with a wizard script

## Usage

Run the wizard script:

```
./terraform.sh
```

Keep deploying !
